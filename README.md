# rich_editor

一款小巧精致、功能丰富的 flutter 富文本编辑器，基于[flutter-quill](https://github.com/singerdmx/flutter-quill)开发.

## ✨ 功能特点

- 支持标题、加粗、链接、撤销、重做等基本功能
- 支持亮色和暗色模式切换
- 支持@用户功能
- 支持高亮显示功能
- 支持输入字符数统计
- 支持自定义工具栏
